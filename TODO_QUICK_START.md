# Todo Management System - Quick Start Guide

## âš¡ 5-Minute Setup for New Users

### Step 1: Access the System
1. Open your web browser
2. Navigate to the Todo system URL (provided by your administrator)
3. Log in with your credentials

### Step 2: First Time Configuration Check
**For Administrators Only:**
1. Go to "Configuration" in the navigation menu
2. Verify these are set up:
   - At least 1 Todo Category (e.g., "General Tasks")
   - At least 1 Priority Level (e.g., "Medium")
   - At least 1 Status (e.g., "Open")
   - At least 1 Location and 1 Branch
   - At least 1 Department

### Step 3: Create Your First Task
1. Click "Task List" in the navigation
2. Click "Add New Todo Task" to expand the form
3. **Quick Method:**
   - Select a job title from the dropdown (if available)
   - Choose requesting branch and department
   - Add a brief task description
   - Click "Add Task"

4. **Manual Method (if no pre-configured tasks):**
   - Fill in all required fields manually
   - Set priority, due date, and assignment

### Step 4: View and Manage Tasks
1. Tasks appear in card format by default
2. Use filters to find specific tasks:
   - Filter by branch, department, or status
   - Toggle "Show Daily Tasks Only" for urgent items
3. Switch between Card and Table views using the toggle buttons

### Step 5: Work with Task Actions
- **For new tasks:** Click "Close" when complete
- **Add comments:** Click "Comments" to discuss or add notes
- **Track progress:** Use status updates and comments
